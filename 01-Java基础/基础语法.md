### 2. **Hello World程序** ⭐⭐⭐⭐⭐

```java

 * 重点：类名必须与文件名一致，main方法是程序入口

    }
```

**重点说明**：
- `public class` - 公共类声明
- `static` - 静态方法，无需创建对象即可调用
- `void` - 无返回值
- `String[] args` - 命令行参数

#### 基本数据类型（8种）

```java
public class DataTypeDemo {
    public static void main(String[] args) {
        // 整数类型
        byte b = 127;           // 1字节，范围：-128~127

        boolean flag = true;    // 只有true/false ⭐⭐⭐⭐⭐

        System.out.println("double: " + d);
        System.out.println("char: " + c);
        System.out.println("boolean: " + flag);
    }
}
```

### 5. **运算符** ⭐⭐⭐⭐⭐

```java
public class OperatorDemo {
    public static void main(String[] args) {
        // 算术运算符
        int a = 10, b = 3;
        System.out.println("加: " + (a + b));      // 13
        System.out.println("减: " + (a - b));      // 7
        System.out.println("乘: " + (a * b));      // 30
        System.out.println("除: " + (a / b));      // 3（整数除法）
        System.out.println("取模: " + (a % b));    // 1 ⭐⭐⭐⭐
        
        // 自增自减
        int x = 5;
        System.out.println("x++: " + (x++));  // 5，先使用后加
        System.out.println("x现在: " + x);     // 6
        System.out.println("++x: " + (++x));  // 7，先加后使用 ⭐⭐⭐⭐
        
        // 比较运算符 ⭐⭐⭐⭐⭐
        System.out.println("a > b: " + (a > b));   // true
        System.out.println("a == b: " + (a == b)); // false
        System.out.println("a != b: " + (a != b)); // true
        
        // 逻辑运算符 ⭐⭐⭐⭐⭐
        boolean condition1 = true;
        boolean condition2 = false;
        System.out.println("与: " + (condition1 && condition2)); // false
        System.out.println("或: " + (condition1 || condition2)); // true
        System.out.println("非: " + (!condition1));              // false
        
        // 三元运算符 ⭐⭐⭐⭐
        int max = (a > b) ? a : b;
        System.out.println("较大值: " + max);
    }
}
```

### 6. **控制流语句** ⭐⭐⭐⭐⭐

#### if-else语句

```java
public class IfElseDemo {
    public static void main(String[] args) {
        int score = 85;
        
        // 单分支
        if (score >= 60) {
            System.out.println("及格");
        }
        
        // 双分支
        if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
        
        // 多分支 ⭐⭐⭐⭐⭐
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else if (score >= 70) {
            System.out.println("中等");
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}
```

#### switch语句

```java
public class SwitchDemo {
    public static void main(String[] args) {
        // 传统switch ⭐⭐⭐⭐
        int day = 3;
        String dayName;
        switch (day) {
            case 1:
                dayName = "星期一";
                break;
            case 2:
                dayName = "星期二";
                break;
            case 3:
                dayName = "星期三";
                break;
            default:
                dayName = "未知";
        }
        System.out.println(dayName);
        
        // Java 14+增强switch（推荐） ⭐⭐⭐⭐⭐
        String result = switch (day) {
            case 1 -> "星期一";
            case 2 -> "星期二";
            case 3 -> "星期三";
            case 4 -> "星期四";
            case 5 -> "星期五";
            case 6, 7 -> "周末";  // 多个case
            default -> "未知";
        };
        System.out.println(result);
    }
}
```

#### 循环语句

```java
public class LoopDemo {
    public static void main(String[] args) {
        // for循环（最常用） ⭐⭐⭐⭐⭐
        System.out.println("for循环：");
        for (int i = 1; i <= 5; i++) {
            System.out.print(i + " ");
        }
        System.out.println();
        
        // 增强for循环（遍历数组/集合） ⭐⭐⭐⭐⭐
        System.out.println("增强for循环：");
        int[] numbers = {10, 20, 30, 40, 50};
        for (int num : numbers) {
            System.out.print(num + " ");
        }
        System.out.println();
        
        // while循环 ⭐⭐⭐⭐
        System.out.println("while循环：");
        int count = 1;
        while (count <= 5) {
            System.out.print(count + " ");
            count++;
        }
        System.out.println();
        
        // do-while循环（至少执行一次） ⭐⭐⭐
        System.out.println("do-while循环：");
        int num = 1;
        do {
            System.out.print(num + " ");
            num++;
        } while (num <= 5);
        System.out.println();
        
        // break和continue ⭐⭐⭐⭐
        System.out.println("break示例：");
        for (int i = 1; i <= 10; i++) {
            if (i == 6) break;  // 跳出循环
            System.out.print(i + " ");
        }
        System.out.println();
        
        System.out.println("continue示例：");
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) continue;  // 跳过本次循环
            System.out.print(i + " ");  // 只打印奇数
        }
        System.out.println();
    }
}
```

### 7. **方法（函数）** ⭐⭐⭐⭐⭐

```java
public class MethodDemo {
    /**
     * 方法定义
     * 访问修饰符 [static] 返回类型 方法名(参数列表) {
     *     方法体
     *     return 返回值;
     * }
     */
    
    // 无参数无返回值
    public static void sayHello() {
        System.out.println("Hello!");
    }
    
    // 有参数无返回值
    public static void greet(String name) {
        System.out.println("你好, " + name + "!");
    }
    
    // 有参数有返回值 ⭐⭐⭐⭐⭐
    public static int add(int a, int b) {
        return a + b;
    }
    
    // 可变参数（Java 5+） ⭐⭐⭐⭐
    public static int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        return total;
    }
    
    // 方法重载（同名不同参） ⭐⭐⭐⭐⭐
    public static int multiply(int a, int b) {
        return a * b;
    }
    
    public static double multiply(double a, double b) {
        return a * b;
    }
    
    public static int multiply(int a, int b, int c) {
        return a * b * c;
    }
    
    public static void main(String[] args) {
        sayHello();
        greet("张三");
        
        int result = add(10, 20);
        System.out.println("10 + 20 = " + result);
        
        // 可变参数调用
        System.out.println("sum(1,2,3) = " + sum(1, 2, 3));
        System.out.println("sum(1,2,3,4,5) = " + sum(1, 2, 3, 4, 5));
        
        // 方法重载调用
        System.out.println("multiply(3,4) = " + multiply(3, 4));
        System.out.println("multiply(3.5,2.5) = " + multiply(3.5, 2.5));
        System.out.println("multiply(2,3,4) = " + multiply(2, 3, 4));
    }
}
```

## 💡 重点总结

1. **数据类型选择**：
   - 整数优先用`int`，超大数用`long`
   - 小数优先用`double`
   - 字符串用`String`
   - 布尔值用`boolean`

2. **命名规范**：
   - 变量/方法：驼峰命名（camelCase）
   - 常量：全大写+下划线（UPPER_CASE）
   - 类名：帕斯卡命名（PascalCase）

3. **控制流**：
   - 优先使用增强for循环遍历集合
   - Java 14+使用新式switch表达式
   - 合理使用break和continue

4. **方法设计**：
   - 单一职责原则
   - 方法名要见名知意
   - 善用方法重载
